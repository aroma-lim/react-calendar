# React Calendar

## Getting Started

To install npm package:

### `npm install`

To execute the program:

### `npm start`

[http://localhost:3000](http://localhost:3000)

## Dependency

- [Node.js](https://nodejs.org/ko/)
- React
- Typescript
- Styled Component
- Redux Toolkit
